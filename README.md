# Hospitality Dashboard Project

## 📊 Project Overview
This project analyzes hospitality and sales data using Excel, Power BI, Tableau, and SQL.

## 🔧 Tools Used
- Excel
- Power BI
- Tableau
- SQL

## 📈 Key Insights
- Total Sales Analysis
- Profit Trends
- KPI Metrics
- Customer Behavior Insights

## 📁 Files Included
- Power BI Dashboard (.pbix)
- Excel Dataset (.xlsx)
- Tableau Dashboard (.twbx)

## 🎯 Objective
To create an interactive dashboard for business decision-making and performance tracking.
